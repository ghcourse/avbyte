# avbyte

A vibrant React + Vite site for a marketing and social networking concept focused on:
- latest market trends
- trending stocks
- crypto news
- AI news

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This repo is configured for GitHub Pages under:

- Repo: `ghcourse/avbyte`
- Base path: `/avbyte/`

After building, deploy the `dist/` output with GitHub Pages or a Pages workflow.

## Concept

avbyte is designed as a bold, premium-feeling market intelligence brand that blends finance, AI culture, and community momentum.
